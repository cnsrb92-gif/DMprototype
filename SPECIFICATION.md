# 배경 및 목적

이돌라 세계관의 캐릭터들과 진짜 메신저처럼 대화하는 경험을 제공한다.
유저는 카카오톡/인스타 DM과 유사한 인터페이스에서 캐릭터에게 메시지를 보내고, 답장을 받고, 인스타 피드에 댓글을 달며 관계를 발전시킨다.

**핵심 경험**: "진짜 사람과 톡하는 것 같은 느낌"
- 캐릭터가 먼저 연락을 걸어온다
- 답장을 안 하면 캐릭터가 읽씹에 반응한다
- 다른 캐릭터 피드에 댓글을 달면 질투 DM이 온다
- 시간대에 따라 캐릭터의 활동과 말투가 달라진다

# 유저 스토리

- As a 유저, I want 캐릭터에게 자유롭게 메시지를 보내고 성격에 맞는 답장을 받고 싶다, so that 진짜 대화하는 느낌을 받을 수 있다.
- As a 유저, I want 캐릭터가 먼저 말을 걸어오게 하고 싶다, so that 내가 신경 쓰이는 존재라는 느낌을 받을 수 있다.
- As a 유저, I want 대화와 행동에 따라 관계가 변하는 걸 눈으로 확인하고 싶다, so that 성장감과 목표 의식을 느낄 수 있다.
- As a 유저, I want 선물을 주거나 데이트를 해서 관계를 더 빠르게 발전시키고 싶다, so that 다양한 상호작용을 즐길 수 있다.
- As a 유저, I want 인스타 피드에 댓글을 달고 캐릭터의 반응을 보고 싶다, so that 채팅 외에도 다양한 소통 채널을 느낄 수 있다.

# 상세 설계

---

## 1. 앱 시작 화면

유저가 앱에 처음 접속하면 **환영 화면**이 나타난다.
Gemini API 키를 입력하고 "시작하기" 버튼을 누르면 채팅 홈으로 이동한다.
한 번 입력한 키는 저장되어 다음 접속부터는 바로 채팅 홈으로 진입한다.

---

## 2. 채팅 홈 화면

카카오톡 채팅 목록과 유사한 화면이다.

### 2.1 스토리 바 (상단)

- 화면 상단에 캐릭터 프로필 원형이 **가로 스크롤**로 나열된다
- 원형을 탭하면 해당 캐릭터의 **인스타 피드 화면**으로 이동한다
- **새 게시글이 올라온 캐릭터**: 원형 테두리가 핑크~보라색 그라데이션 링으로 빛나고, 빨간 숫자 뱃지(새 게시글 수)가 표시된다
- **새 게시글이 없는 캐릭터**: 회색 테두리
- 맨 앞에 "ME" 버튼이 있으며, 탭하면 **내 프로필 화면**으로 이동한다
- 스토리 원형은 **새 게시글이 있는 캐릭터가 앞쪽**에 배치된다

### 2.2 채팅 목록 (하단)

- 각 캐릭터의 아바타, 이름, 마지막 메시지 미리보기, 시간이 표시된다
- 안 읽은 메시지가 있으면 **초록색 뱃지**로 안 읽은 개수가 표시된다
- **가장 최근에 메시지를 받은 캐릭터가 맨 위**에 정렬된다
- 캐릭터를 탭하면 **채팅방**으로 진입한다

---

## 3. 채팅방

### 3.1 헤더 영역

| 요소 | 설명 |
| --- | --- |
| 뒤로 버튼 | 채팅 홈으로 돌아간다 |
| 캐릭터 아바타 + 이름 | 대화 상대 표시 |
| 관계 레벨 뱃지 | 현재 관계 단계를 색상 라벨로 표시 (예: 보통 🙂, 썸 💘) |
| 텐션 게이지 | 핑크색 프로그레스 바. 가득 차면 하트가 두근두근 뛴다 |

### 3.2 대화 영역

- 유저 메시지는 **오른쪽**, 캐릭터 메시지는 **왼쪽**에 말풍선으로 표시된다
- 캐릭터가 답장할 때 메시지가 **한 번에 뜨지 않고 여러 말풍선으로 나뉘어** 뚝뚝 도착한다
  (예: "밥 먹었어?" → 0.7초 후 → "나 지금 치킨 먹는 중ㅋㅋ" → 0.8초 후 → "부럽지?")
- 선물이나 데이트 같은 이벤트는 **시스템 메시지**(회색 안내문)로 표시된다
- 새 메시지가 오면 자동으로 **하단 스크롤**된다

### 3.3 하트 파티클 연출

대화 중 **텐션 게이지가 올라가면**:
- 채팅방 안에 **핑크색 하트**들이 아래에서 위로 둥실둥실 떠오른다
- 변화량이 클수록 하트가 많이 뜬다 (최대 6개)
- 하트마다 크기와 위치가 조금씩 다르고, 시간차를 두고 하나씩 올라온다

**텐션 게이지가 내려가면**:
- **회색 깨진 하트**들이 위에서 아래로 우수수 떨어진다
- 떨어지면서 약간 기울어지는 연출이 있다

### 3.4 선물 보내기

채팅 입력란 옆 선물 버튼을 누르면 **선물 메뉴**가 펼쳐진다.

| 선물 | 눈에 보이는 효과 |
| --- | --- |
| 치킨 🍗 | 텐션 게이지가 조금 오른다. 캐릭터가 좋아하는 반응을 보낸다 |
| 명품 가방 💎 | 텐션 게이지가 크게 오른다. 캐릭터가 놀라며 반응한다 |
| 아파트 🏢 | 관계도가 크게 오른다. 캐릭터가 감동하는 반응을 보낸다 |

선물 후 3초간은 추가 선물을 보낼 수 없다 (연타 방지).

### 3.5 데이트 버튼

텐션 게이지가 **100%에 도달**하면:
- 채팅방 하단에 **통통 튀는 데이트 버튼**이 나타난다
- 탭하면 랜덤 데이트 시나리오가 선택된다 (한강 치맥, 놀이공원, 영화관, 카페 중 하나)
- 데이트가 끝나면:
  - **"데이트 종료!" 팝업**이 뜨고, "후일담 듣기" 버튼을 누르면
  - 채팅에 데이트 후기 메시지와 **포토카드 이미지**가 추가된다
  - 캐릭터의 인스타에 **데이트 게시글**이 자동으로 올라간다
  - 텐션 게이지는 **0으로 리셋**되어 다시 채우기 시작한다

## 4. 캐릭터 시스템

### 4.1 등장 캐릭터

**앤** — 버밀리온 제국 제1황녀
- 성격: 활발하고 에너지 넘침 (골든 리트리버 같은 성격)
- 말투: 반말, 애교 섞인 직설적인 표현
- 특징: 가장 적극적으로 먼저 말을 건다 (60% 확률)

**네라** — 귀차니즘 뱀파이어
- 성격: 귀찮아하면서도 신경 쓰는 츤데레
- 말투: 반말, 귀찮은 듯한 톤, 짧은 문장
- 특징: 좀처럼 먼저 연락하지 않는다 (30% 확률)

**소피아** — 모델 겸 대학생
- 성격: 다정하지만 질투가 많은 얀데레 기질
- 말투: 존댓말과 반말을 부드럽게 섞어 쓴다
- 특징: 유저의 행동에 민감하게 반응한다 (50% 확률)

### 4.2 시간대별 에너지

캐릭터의 응답 스타일이 **현실 시간**에 따라 달라진다.

| 시간대 | 에너지 | 캐릭터 반응 |
| --- | --- | --- |
| 오전 10시 ~ 오후 2시 | 높음 | 말이 많아진다. 메시지를 여러 개로 나눠 보낸다 |
| 오전 7시 ~ 오전 10시, 오후 2시 ~ 밤 10시 | 보통 | 평소와 같은 톤으로 대화한다 |
| 밤 10시 ~ 아침 7시 | 낮음 | 한 줄짜리 짧은 답만 한다. 졸린 톤 |

### 4.3 시간표 (캐릭터별 하루 일과)

각 캐릭터는 시간대별로 다른 활동을 하고 있다.
캐릭터가 먼저 보내는 메시지 내용이 현재 활동에 따라 달라진다.

**앤의 하루 (예시)**:
- 아침 7~9시: 산책 → "산책 중인데 너도 같이 왔으면!"
- 오전 9~12시: 수업 → "수업 끝나면 놀자~!"
- 점심 12~2시: 자유시간 → "밥 뭐 먹을 거야?"
- 오후 2~6시: 모험 → "오늘 대박 사건 있었어!!"
- 저녁 6~9시: 휴식 → "오늘 하루 어땠어~?"
- 밤 9~11시: 일기 쓰기 → "오늘 일기에 뭐 쓰지..."
- 밤 11시~아침 7시: 수면 → "zzz..."

## 5. 관계도 시스템

### 5.1 관계 레벨

캐릭터와의 관계가 5단계로 나뉜다. 채팅방 헤더와 프로필에서 확인할 수 있다.

| 단계 | 이름 | 설명 |
| --- | --- | --- |
| 1단계 | 혐오 👿 | 캐릭터가 유저를 싫어하는 상태. 차가운 반응 |
| 2단계 | 무관심 😐 | 관심이 없는 상태. 대충 대답한다 |
| 3단계 | 보통 🙂 | 평범한 대화 상대. 나쁘지 않은 관계 |
| 4단계 | 친함 🥰 | 친밀한 관계. 말투가 부드러워진다 |
| 5단계 | 썸 💘 | 연애 감정이 생긴 상태. 설레는 대화 |

### 5.2 관계가 변하는 순간

- **올라가는 경우**: 좋은 대화를 하거나, 피드에 댓글을 달거나, 아파트를 선물하면 올라간다
- **내려가는 경우**: 캐릭터의 메시지를 무시(읽씹)하거나, 오래 답장을 안 하면 내려간다
- 관계 레벨이 변경되면 **레벨업 모달**이 팝업된다
  - 캐릭터 특별 일러스트와 함께 "관계 발전!" 메시지가 표시된다
  - 이전 단계 → 새 단계가 나란히 보여진다

### 5.3 초기 관계 상태

게임 시작 시 캐릭터마다 관계도가 다르게 설정되어 있다.

| 캐릭터 | 시작 관계 | 설명 |
| --- | --- | --- |
| 앤 | 보통 🙂 (45/100) | 이미 알고 지내는 사이 |
| 네라 | 혐오 👿 (10/100) | 귀찮은 존재로 보고 있다 |
| 소피아 | 썸 💘 (90/100) | 이미 호감이 깊은 상태. 데이트 직전 |

## 6. 데이트 텐션 시스템

### 6.1 텐션 게이지란?

채팅방 헤더 아래에 **핑크색 프로그레스 바**가 있다.
이것이 "데이트 텐션"으로, 캐릭터와의 두근거림 수치다.
**100%가 되면 데이트를 신청할 수 있다.**

### 6.2 텐션이 변하는 순간

**올라가는 경우**:
- 캐릭터와 좋은 대화를 했을 때 (AI가 판단)
- 치킨 선물 → 조금 오름
- 명품 가방 선물 → 크게 오름
- 인스타 댓글 → 소폭 오름

**내려가는 경우**:
- 메시지를 읽고도 답장을 안 했을 때
- 캐릭터의 선톡을 3개 이상 무시했을 때
- 데이트를 완료하면 → **0으로 초기화** (다시 채우기 시작)

### 6.3 100% 도달 시

- 텐션 게이지의 하트가 **두근두근 뛰는 애니메이션**으로 바뀐다
- 채팅방에 **데이트 버튼이 통통 튀며** 나타난다
- 탭하면 데이트가 시작된다

## 7. 선톡(먼저 연락) 시스템

### 7.1 개요

유저가 가만히 있어도 캐릭터가 **먼저 메시지를 보낸다.**
마치 진짜 상대방이 연락을 걸어오는 것 같은 경험을 제공한다.

### 7.2 선톡이 오는 상황

| 상황 | 설명 |
| --- | --- |
| 채팅방을 나갈 때 | 다른 캐릭터들이 "나도 있는데?" 하며 말을 건다 |
| 5초간 아무 입력이 없을 때 | 다른 캐릭터가 틈을 타서 메시지를 보낸다 |
| 캐릭터 활동이 바뀔 때 | 1분마다 확인하며, 활동이 바뀌면 관련 메시지를 보낸다 |

### 7.3 선톡이 올 때 유저에게 보이는 것

1. 화면 상단에 **토스트 알림**이 내려온다 (캐릭터 아바타 + 메시지 미리보기)
2. 알림음이 울린다
3. 채팅 홈의 해당 캐릭터 칸에 **안읽음 뱃지 숫자**가 올라간다
4. 토스트를 탭하면 해당 채팅방으로 바로 이동한다
5. 3초 후 토스트가 자동으로 사라진다

### 7.4 무시했을 때 반응 변화 (에스컬레이션)

캐릭터의 선톡에 답장을 안 하면, 다음 선톡의 톤이 점점 변한다.

| 무시 횟수 | 앤 (예시) | 네라 (예시) | 소피아 (예시) |
| --- | --- | --- | --- |
| 0~1회 | "뭐해?" "나 심심해!" | "..." "뭐" | "있어요? 🥺" |
| 2~4회 | "야 왜 씹어 😡" | "하… 바쁜가 보네" | "혹시 바쁘신 건가요...?" |
| 5회 이상 | "나 싫어졌어...?" | "알았어 그냥" | "제가 뭔가 잘못했나요..." |

## 8. 읽씹 감지 시스템

### 8.1 개요

유저가 채팅방에 들어와서 **메시지를 읽은 뒤 답장을 안 하면**, 캐릭터가 반응한다.
"읽고 씹었다"는 걸 캐릭터가 알아채는 것이다.

### 8.2 동작 방식

1. 채팅방에 들어가면 30초 카운트가 시작된다
2. 30초 동안 아무것도 입력하지 않으면, 캐릭터가 **첫 번째 읽씹 반응**을 보낸다
3. 이후 30초마다 반응이 **점점 감정적**으로 변한다 (총 6단계)
4. 유저가 메시지를 보내는 순간 읽씹 카운트가 **즉시 중단**된다

### 8.3 단계별 반응 예시

| 단계 | 앤 | 네라 | 소피아 |
| --- | --- | --- | --- |
| 30초 | 짜증 (재촉) | 짜증 | 걱정하며 부탁 |
| 1분 | "보고 있으면서!!" | "읽고 씹는 거지?" | "왜 안 답해요..?" |
| 1분 30초 | 걱정으로 전환 | 의심 시작 | 불안해하기 시작 |
| 2분 | 서운해짐 | "궁금하지도 않지?" | 자책하기 시작 |
| 2분 30초 | 자기 탓으로 돌림 | 후회하는 투 | "미안해요..." |
| 3분 | 외로워함 | 포기하는 투 | 애원에 가까운 반응 |

### 8.4 페널티

- 첫 번째 읽씹 감지 시: **관계도 소폭 하락 + 텐션 소폭 하락**
- 이후 단계에서는 추가 페널티 없이 감정 메시지만 전송된다

---

## 9. 인스타 피드 시스템

### 9.1 개요

각 캐릭터가 자체 **인스타그램 피드**를 운영한다.
채팅 홈의 스토리 원형을 탭해서 진입한다.

### 9.2 게시글 업로드 타이밍

캐릭터들은 **자동으로 피드를 올린다**:
- 활동이 바뀔 때 (일정 확률로)
- 20~30분마다 (상태 게시글)
- 데이트 후 (데이트 후기 게시글)

새 게시글이 올라오면 채팅 홈의 스토리 원형에 **그라데이션 링 + 빨간 뱃지**로 알려준다.

### 9.3 게시글 구성

하나의 게시글에는 다음이 표시된다:
- 캐릭터 **포토카드 이미지**
- **캡션** (현재 활동에 맞는 한 줄 글)
- **좋아요 수** (랜덤 생성)
- **타임스탬프** ("방금", "1시간 전" 등)
- **다른 캐릭터들의 댓글** (AI가 생성)

### 9.4 댓글 달기

유저가 댓글을 달면 다음이 일어난다:

1. **즉시 반응**
   - 유저의 댓글이 게시글에 표시된다
   - 관계도가 올라가고, 텐션도 소폭 오른다
   - 경험치를 획득한다

2. **캐릭터 답글** (몇 초 후)
   - 피드 주인 캐릭터가 AI로 생성된 답글을 단다
   - 예: 유저가 "이쁘다" → 앤: "ㅋㅋ 당연하지~"

3. **다른 캐릭터의 질투 반응** (확률적)
   - 다른 캐릭터가 해당 게시글에 **질투 섞인 댓글**을 달 수 있다
   - 또는 유저에게 **질투 DM**을 보낼 수 있다
     - 예: 소피아 피드에 댓글을 달았더니, 앤이 DM으로 "야 너 소피아한테 뭐라고 한 거야?" 라고 보낸다
   - 질투의 강도는 해당 캐릭터와의 **현재 관계 단계**에 따라 달라진다

### 9.5 댓글과 좋아요 순서

- 좋아요 버튼은 **댓글을 먼저 달아야** 누를 수 있다
- 댓글 전에 좋아요를 누르면 "댓글을 먼저 달아보세요!" 넛지가 표시된다

---

## 10. 선물 시스템

### 10.1 개요

채팅방에서 캐릭터에게 선물을 보낼 수 있다.
채팅 입력란 옆 선물 아이콘을 탭하면 선물 메뉴가 열린다.

### 10.2 선물 종류와 효과

| 선물 | 채팅에 보이는 것 | 효과 |
| --- | --- | --- |
| 치킨 🍗 | "🍗 치킨 기프티콘을 보냈습니다." | 텐션 소폭 상승. 캐릭터가 좋아하는 반응 |
| 명품 가방 💎 | "💎 명품 가방을 보냈습니다." | 텐션 크게 상승. 캐릭터가 놀라는 반응 |
| 아파트 🏢 | "🏢 강남 아파트를 선물했습니다!!!" | 관계도 크게 상승. 캐릭터가 감동 반응 |

### 10.3 동작

1. 선물 버튼 탭 → 3개 선물 중 선택
2. 채팅에 시스템 메시지가 표시된다
3. 캐릭터가 AI로 생성된 **선물 리액션 메시지**를 보낸다
4. 관계도/텐션이 변하고, 경험치를 얻는다
5. 선물 후 **3초간 쿨다운** (다음 선물 불가)

---

## 11. 데이트 시스템

### 11.1 발동 조건

텐션 게이지가 100%에 도달하면 채팅방에 **데이트 버튼**이 나타난다.

### 11.2 시나리오

4종류 중 랜덤 선택:
- 한강에서 치맥 먹기 🍗🍺
- 놀이공원 데이트 🎡
- 영화관 데이트 🍿
- 분위기 좋은 카페 ☕

### 11.3 데이트 진행 흐름

1. 데이트 버튼 탭
2. **데이트 종료 팝업**이 뜬다 (하트 아이콘 + 시나리오 이름)
3. "후일담 듣기" 탭 →
   - 채팅에 데이트 관련 메시지가 추가된다
   - 캐릭터의 **랜덤 포토카드**가 채팅에 표시된다
   - 캐릭터 인스타에 **데이트 게시글**이 자동으로 올라온다
   - 다른 캐릭터들이 해당 게시글에 댓글을 단다
   - 텐션 게이지가 **0으로 리셋**된다
   - 경험치 +20

## 12. 플레이어 프로필

### 12.1 접근 방법

채팅 홈의 스토리 바에서 "ME" 버튼을 탭한다.

### 12.2 프로필 카드

| 항목 | 내용 |
| --- | --- |
| 이름 | 의뢰인 |
| 직함 | 뭐든집 운영자 |
| 소개 | 이돌라에서 만능 해결 사무소 '뭐든집'을 운영하는 청년 |
| 좌우명 | 선행은 선과로, 악행은 악과로 돌아온다 |
| 성격 태그 | 성실, 원칙적, 통찰력, 적응력, 츤데레 |

### 12.3 레벨 시스템

경험치(XP)를 모아 레벨업할 수 있다. 프로필 화면에 프로그레스 바로 표시된다.

| 레벨 | 칭호 | 필요 XP |
| --- | --- | --- |
| 1 | 견습 의뢰인 | 0 |
| 2 | 초보 해결사 | 50 |
| 3 | 숙련된 해결사 | 150 |
| 4 | 뭐든집 단골 | 300 |
| 5 | 뭐든집 VIP | 500 |

**XP 획득 방법**:
- 메시지 전송 → +2
- 선물 전달 → +5
- 피드 댓글 → +10
- 데이트 완료 → +20

### 12.4 관계 현황

프로필 하단에 **각 캐릭터별 관계 카드**가 표시된다:
- 캐릭터 아바타 + 이름
- 현재 관계 레벨 뱃지
- 관계도 수치 (0~100) 프로그레스 바
- 텐션 수치 프로그레스 바
- 텐션이 100이면 "데이트 가능!" 라벨 표시

---

## 13. 토스트 알림

### 13.1 동작

다른 채팅방에 있거나 채팅 홈에 있을 때, 캐릭터가 메시지를 보내면:
- 화면 **최상단에 알림 배너**가 위에서 내려온다
- 캐릭터 아바타 + 메시지 미리보기 + "보기" 링크가 표시된다
- 알림을 탭하면 해당 채팅방으로 바로 이동한다

### 13.2 스택 표시

- 알림이 여러 개 동시에 쌓이면 **겹쳐서 표시**된다 (최대 3개)
- 아래 알림일수록 약간 작고 투명하게 보인다
- 3초 후 자동으로 위로 사라진다
